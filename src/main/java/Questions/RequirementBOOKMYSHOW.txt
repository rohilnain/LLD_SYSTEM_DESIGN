                                            BookMyShow/Ticket Master System
1.It should be able to list the cities where affiliate cinemas are present.
2.Each cinema can have multiple audi's and each audi can run one movie at a time.
3.Each Movie will have multiple shows.
4.Customers should be able to search movies by their title, language, genre, release date, and city name.
5.Once the customer selects a movie, the service should display the cinemas running the movie and its available shows.
6.The customer should be able to select a show at a particular cinema and book their tickets.
7.The service should show the customer the seating arrangement of the cinema hall.The customer should be able to select
  multiple seats according to their preference.
8.The customer should be able to distinguish between available seats and booked ones.
